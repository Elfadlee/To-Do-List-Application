React To-Do List App

This is a simple To-Do List application built with React and Material UI (MUI).
The app provides an intuitive interface for managing daily tasks with a clean and modern user experience.

Features

Add new tasks with a title and description

Mark tasks as completed or uncompleted with clear visual feedback

Edit existing tasks using a clean Material UI dialog

Delete tasks with a confirmation dialog

Snackbar notifications for task actions (add, edit, delete, status updates)

LocalStorage persistence to keep tasks after page reload

Modern UI design using Material UI, including Floating Action Buttons and custom colors

Getting Started

Install dependencies:

npm install

Start the development server:

npm start

Open the application in your browser:

http://localhost:3000
